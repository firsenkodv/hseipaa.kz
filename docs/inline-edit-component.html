x-admin.inline-edit

Компонент фронтендного редактирования HTML-полей без входа в админ-панель.
Видим только залогиненному администратору MoonShine.

Обзор

Оборачивает любой блок контента. При наведении администратор видит кнопку-карандаш. Клик открывает модалку с <textarea> и исходным HTML. После сохранения контент обновляется прямо в DOM — без перезагрузки страницы.

Файлы

  • 🐘 app/Http/Controllers/Admin/InlineEditController.php Backend: авторизация, whitelist, сохранение
  • 🔀 routes/web.php POST /admin-inline-edit
  • 🖼 resources/views/components/admin/inline-edit.blade.php Blade-компонент
  • 🎨 resources/css/components/admin/inline-edit.scss Стили кнопки и модалки
  • ⚡ resources/js/include/site/inline-edit.js Vanilla JS: модалка, fetch, обновление DOM

Подключение

app.css

@import 'components/admin/inline-edit.scss';

script.js

import { initInlineEdit } from "./include/site/inline-edit";

document.addEventListener('DOMContentLoaded', () => {
    initInlineEdit();
});

routes/web.php

Route::post('/admin-inline-edit', [InlineEditController::class, 'update'])
    ->name('admin.inline-edit');

Использование

<x-admin.inline-edit :model="$item" field="desc" label="Описание">
    <div class="content_page__desc desc">
        {!! $item->desc !!}
    </div>
</x-admin.inline-edit>

СохранитьPOST /admin-inline-edit

DOM updateчерез 1 сек, без reload

Модель и поле передаются через data-атрибуты кнопки. JS читает их, формирует JSON-запрос с CSRF-токеном из <meta name="csrf-token"> и обновляет innerHTML первого дочернего элемента контентного блока.

Безопасность

auth('moonshine')

Проверка на бэкенде при каждом запросе. Незалогиненный получает 403.

Whitelist моделей — $modelMap

Клиент передаёт alias (training, news…). Контроллер маппит на класс. Произвольный класс передать невозможно.

Whitelist полей — $allowedFields

Можно сохранять только в разрешённые поля. Запись в password, slug и другие невозможна.

Расширение whitelist

Новая модель

protected array $modelMap = [
    // ... существующие ...
    'page' => \App\Models\Page::class,
];

Новое поле

protected array $allowedFields = [
    'desc', 'html', 'desc2', 'html2', 'short_desc',
    'content', // добавить
];
