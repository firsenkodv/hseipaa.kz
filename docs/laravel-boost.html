Laravel Boost — Установка и Skills

Шаг 1. Установка пакета

composer require laravel/boost --dev

Шаг 2. Инициализация

php artisan boost:install

Появится интерактивное меню с выбором фич. Нужно вручную ввести нужные значения через запятую:

guidelines,skills,mcp

Или только нужное, например:

skills

Шаг 3. Установка skill

php artisan boost:add-skill <owner/repo> --skill <skill-name>

Пример:

php artisan boost:add-skill jeffallan/claude-skills --skill laravel-specialist

Шаг 4. Директория skills

Все доступные skills: skills.laravel.cloud

Важно: boost:add-skill — это отдельная команда в терминале. Не вводить её в меню boost:install.
